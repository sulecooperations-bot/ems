# Brand Assets

## Logo

The SULECO logo should be placed here:

- **File**: `suleco-logo-white.png`
- **Source**: https://sulecoltd.com/wp-content/uploads/2024/04/LogoWhite.png
- **Usage**: Used in the sidebar header and other brand locations

## Fallback

A simple "S" mark SVG is available at `suleco-s-mark.svg` as a fallback.

## Download Instructions

If the logo is not present, you can download it manually:

```bash
curl -o public/brand/suleco-logo-white.png https://sulecoltd.com/wp-content/uploads/2024/04/LogoWhite.png
```

Or download it from the URL above and place it in this directory.
